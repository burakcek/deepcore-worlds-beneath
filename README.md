# DeepCore: Worlds Beneath

An original, playable browser prototype for a futuristic voxel-exploration game. It is deliberately inspired by the *sandbox genre*, not Minecraft's characters, branding, art, code, or assets.

## Run it

Open `index.html` in a modern browser, or publish this folder using GitHub Pages.

## Controls

- **A/D** or arrow keys: move
- **W**, up arrow, or space: jump
- **Hold click**: operate the plasma drill

Drilling collects ferrite, crystal cores, and coolant. Drilling continuously increases heat; give the tool a moment to cool down when it reaches maximum.

Create a new GitHub repository, upload these files, then enable **Settings → Pages → Deploy from a branch** and select the branch containing `index.html`.
